# Git Status Checklist

Use this checklist before running undo commands.

## Steps

1. Run `git status` to see staged, unstaged, and untracked files.
2. Run `git diff` to inspect unstaged changes.
3. Run `git diff --cached` to inspect staged changes.
4. Confirm which file or commit should be changed.
5. Choose the smallest undo command that fixes the problem.

## Reminder

If you are unsure, save a copy of important work before discarding changes.
